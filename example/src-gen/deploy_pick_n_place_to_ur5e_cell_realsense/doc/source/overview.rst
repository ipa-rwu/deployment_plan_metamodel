############################
Deployment overview
############################

    * color_pose_estimation
    * robot_skills
    * realsense2_camera_driver
    * ur_robot_driver
  to a target environment including devices as follows:

  * rwu_laptop (type: PC)
  * arm (type: UR5E)
  * gripper (type: EGP50)
  * camera_realsense (type: Realsense_camera)
